### 微信公众号接口

首页统计接口：

```
URL：/wechat/count
请求方式：get
请求参数：无
```

企业查询：

```
URL:/wechat/query/company
请求方式：get
请求参数：
	id: 	int		为0时表示首页和列表页面查询,否则为查询某一个具体的企业信息（id为0时后面的参数无效）
	name:	str	搜索关键字
	type:	int	企业类型
	status: int		企业状态
	pid:	int		省
	cid:	int		市
	did:	int		区
	time:	str		查询起始时间
	page:	int		页码
	
返回值：
	id为0时：
	{
		company: list 列表,
		total: int	总数,
		alarm: int	告警
	}
	id为1时：
	{
		
	}
```
