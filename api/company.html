公司注册接口:已完成

URL：/company/regist;

请求方式：POST;

请求参数：{
    Name: 公司名字 str
    Legal: 法人（企业负责人） str
    Address: 公司地址 str
    Phone: 联系方式 list [{name:'', post:'',phone:''}]
    License: 公司资质图片 filelist
    Type: 公司类型 int
    ProvinceID: 所属省ID int
    CityID: 所属市ID int
    DistrictID: 所属区县ID int
    BadRecord: 不良记录 str
    HasBadRecord: 是否有不良记录 bool
    Description: 备注 str
    group_list: 分组ID列表 list (没有则为空列表)
    pic_list: 上传的所有图片ID列表 list （没有则为空列表）
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}
