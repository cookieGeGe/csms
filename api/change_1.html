版本一修改

企业相关：

创建企业接口：/company/regist

更新企业接口：/company/update

请求方式：post;

请求改变参数：
-    BadRecord
+    otherfile list 其他类型的文件列表, 如果没有就传入一个空列表
    HasBadRecord int 值改为1或者2,或者3, 正常：1， 风险：2 黑名单：3

获取企业详情：/company/info

请求方式：get;

返回结果：
+    other_file_list list 其他类型的文件列表, 如果没有就传入一个空列表,返回多个结构体{id: , companyid: filename: ,filepath: }
    HasBadRecord int 值改为1或者2,或者3, 正常：1， 风险：2 黑名单：3
-    BadRecord

上传图片列表：/company/uploadgrouppic

请求方式：post;

请求参数：
    file: list 上传的图片文件列表

返回结果：
+    data list 其他类型的文件列表, 如果没有就传入一个空列表,返回多个结构体{id: 图片id, name:图片名字, url:图片路径 }
-    id
-    name
-    url

上传图片列表：/company/remove/otherfile

请求方式：delete;

请求参数：
    id: 要删除的文件ID

项目相关：

创建项目接口：/project/create

更新项目接口：/project/update

请求方式：post;

请求改变参数：
-    LaborManager
-    Supervisor
    ConsManager list 改为多个负责人的名字列表
    OwnerManager list 改为多个负责人的名字列表
    Duration datetime 越来的工期月数字段，改为时间对象，格式'2019-01-05'

请求参数：

-    id
-    name
-    url

获取项目详情：/project/query

请求方式：get;

返回结果：
+    BuildName str 建设方名称
+    ConsName str 施工方名称

获取项目进度详情：/project/progress/query

请求方式：get;

返回结果：
+    labor_total int 劳工总数
+    attend_percent float 打卡比例
+    bank_should_pay float 银行应付
+    bank_real_pay float 银行实际支付
+    bank_status 银行支付状态（对应银行管理的支付状态）

获取项目列表：/project/search

请求方式：get;

返回结果：
-    LaborManager
-    Supervisor
    WagePercent 工资发放
