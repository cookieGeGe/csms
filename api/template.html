创建和更新模板：已完成

URL：/template/create;

请求方式：GET;

请求参数：{
    ID: 模板ID int(新建的时候不上传ID，编辑的时候才上传这个选项)
    Name: 模板名字 str
    Description: 模板秒速 str
    Type: 模板类型 int (0:企业管理，1：项目管理，2：劳工管理)
    File: 图片文件 file
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

删除模板：已完成

URL：/template/delete;

请求方式：delete;

请求参数：{
    ID: 模板ID int
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

获取单条模板内容：已完成

URL：/template/query/one;

请求方式：get;

请求参数：{
    ID: 模板ID int
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!', 'template':{ID:'', Name:'',Time:'',Number:'',File:'图片路径'， Type:''}

模板搜索：已完成

URL：/template/query;

请求方式：get;

请求参数：{
    Name: 模板名字 str
     Number 模板编号 str
    Type: 模板类型 int (0:企业管理，1：项目管理，2：劳工管理, 3:表示没有选择)， 默认填3
    Time: 模板创建时间 str
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!', 'template_list':[{ID:'', Name:'',Time:'',Number:'',File:'图片路径'， Type:''}]}
