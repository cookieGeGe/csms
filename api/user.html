注册接口：

URL：/user/regist;

请求方式：POST;

请求参数：{
    LoginName：'登录名称 str',
    UserName： '用户名称 str',
    Email:'邮箱 str',
    Password:'密码 str',
    Phone: '电话 str',
    Description:'描述 str ' ,
    CompanyID： '公司ID int',
    AdminType: '管理员类型 int',
AreaID: '区域ID， int',
} 头像文件（如果有的话，可以没有）

失败返回值：{'code': 50006, 'msg': '该用户已存在！'}
{'code': 50003, 'res': 0, 'msg': '存在必填字段为空，请填写必填字段！'}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

登录接口：

URL：/user/login;

请求方式：POST;

请求参数：{
    LoginName：'登录名称 str',
    Password:'密码 str',
}

失败返回值：{'code': 50007, 'msg': '用户名不存在，请输入正确的用户名！'};
    {'code': 50008, 'msg': '密码错误！'};
    {'code': 50010, 'msg': '用户已停用，请联系管理员！'};
    {'code': 50003, 'res': 0, 'msg': '存在必填字段为空，请填写必填字段！'}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

退出登录接口：

URL：/user/logout;

请求方式：GET;

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

删除用户接口：

URL：/user/delete;

请求方式：DELETE;

请求参数： {
    ID: int 用户userID
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

用户停用接口：

URL：/user/down;

请求方式：GET;

请求参数： {
    ID: int 用户userID
}

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}
