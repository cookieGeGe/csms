新建项目：已完成

URL：/project/create;

请求方式：POST;

请求参数：{
    Name: 项目名称 str
    Type: 项目类型 int 下拉框的次序，从0开始依次对应
    GuaranType: 担保类型 int
    Price: 中标价格 str
    Daration: 项目工期（月） int
    GAmount: 担保金额 str
    PrinPical: 项目委托人 [{name:'', post:'', phone:''}] 如果没有则为空''
    WagePercent: 农民工工资占比 str
    StartTime: 开始时间 datetime
    EndTime: 结束时间 datetime
    Address: 地址 str
    Build: 建设方公司ID int
    Cons: 施工方公司ID int
    ConsManager: 施工方负责人 {id:'公司ID'， name:'公司名字', pname：'负责人名字', post: '职位'， phone:'负责人电话'}
    OwnerManager: 业主方负责人 {id:'公司ID'， name:'公司名字', pname：'负责人名字', post: '职位'， phone:'负责人电话'}
    LaborManager: 劳务专员 {id:'公司ID'， name:'公司名字', pname：'负责人名字', post: '职位'， phone:'负责人电话'}
    Supervisor: 监理负责人 {id:'公司ID'， name:'公司名字', pname：'负责人名字', post: '职位'， phone:'负责人电话'}
    Description: 备注 str
    Status: 状态 int
    PID: 省ID int
    CID: 市ID int
    DID: 区县ID int
    Total: 总产值 str
    TotalPay: 总领款 str
    Issue: 发放统计 str
    Group_list: 创建的图片分组的ID列表 [id1,id2]
    Img_list: 上传的图片的ID列表 [id1,id2]
}

以上参数如无说明或者格式之外的限制的字段为空是为null

成功返回值：{'code': 0, 'res': 0, 'msg': '请求成功!'}

失败返回值：{'code': 500xx, 'res': 0, 'msg': '错误消息！'}
